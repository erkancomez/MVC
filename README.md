# MVC

Nuget Paketleri: 

Entities katmanı için: 
Microsoft.AspNetCore.Identity.EntityFrameworkCore

DataAccess Katmanı için:
Microsoft.AspNetCore.Identity
Microsoft.EntityFrameworkCore.Design
Microsoft.EntityFrameworkCore.SqlServer
Microsoft.EntityFrameworkCore.Tools

Business Katmanı için:
FluentValidation.AspNetCore
NLog
NLog.Schema
EPPlus
EPPliTextSharp.LGPLv2.Coreus
FastMember

Projede bootstrap kütüphanesi kullanıldı. 
Kullanılan template: https://bootswatch.com/united/
